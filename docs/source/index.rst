Introduction to MPI 
===================================
.. admonition:: Overview

    * **Tutorial:** 3 hours

            #. Learn what is MPI

**MPI** MPI stands for Message Passing Interface. It is the de-facto standard for parallel programming that enables the program to use large numbder of processors to solve a problem.
